# 📈 Sales Prediction Using Python

This project analyzes advertising data and predicts product sales using Python and Machine Learning techniques.

---

# 🎯 Project Objective

The main objective of this project is to study how advertising budgets on different platforms affect product sales and identify the best prediction model.

---

# 📁 Dataset Information

Dataset contains advertising budgets for:

- 📺 TV Advertising
- 📻 Radio Advertising
- 📰 Newspaper Advertising

Target Variable:
- 💰 Sales

---

# 🤖 Machine Learning Models Used

- 🔹 Linear Regression
- 🔹 Random Forest Regressor

---

# ⚙ Project Workflow

1. Import Dataset
2. Data Cleaning
3. Remove Unnecessary Columns
4. Check Missing Values
5. Feature Selection
6. Train-Test Split
7. Train Machine Learning Models
8. Evaluate Model Performance
9. Compare Model Accuracy
10. Visualize Sales Analysis

---

# 🛠 Technologies Used

- 🐍 Python
- 📘 Pandas
- 🔢 NumPy
- 📉 Matplotlib
- 🌈 Seaborn
- 🤖 Scikit-learn

---

# 📊 Visualizations Included

- ✔ Actual vs Predicted Sales Graph
- ✔ Correlation Heatmap
- ✔ TV Advertising Analysis
- ✔ Sales Distribution Analysis

---

# 📈 Model Evaluation

Models are evaluated using:

- ✔ Mean Absolute Error (MAE)
- ✔ R2 Score

---

# 🏆 Best Model Result

✅ Random Forest Regressor achieved better prediction accuracy compared to Linear Regression.
